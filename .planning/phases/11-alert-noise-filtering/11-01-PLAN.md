# Plan 11-01: Alert Noise Filtering

Add `in_stock` gating to the Telegram alert pipeline to suppress false-positive alerts
for out-of-stock products.

## Requirements

- ALERT-01: Price-drop alerts only fire for in-stock products
- ALERT-02: High-memory alerts only fire for in-stock products

## Tasks

<read_first>
- `src/apple_deals/alerts/telegram.py` — `send_alert()` function
- `src/apple_deals/alerts/telegram.py` — `_format_message()` function
</read_first>

<action>
In `send_alert()`, add a guard at the start:

```python
def send_alert(data: ProductData, old_price: float, new_price: float) -> bool:
    # Suppress alerts for out-of-stock products (false positives)
    if not data.get("in_stock", True):
        return False
    if not should_alert(old_price, new_price):
        return False
```
