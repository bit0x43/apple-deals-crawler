The new crawl() function:

```python
@app.command()
def crawl() -> None:
    """Crawl product prices from all configured stores."""
    from apple_deals.crawlers.mac_center import MacCenterCrawler
    from apple_deals.crawlers.tiendasishop import TiendasishopCrawler
    from apple_deals.db.crud import upsert_if_changed

    crawlers: list[tuple[str, object]] = [
        ("tiendasishop", TiendasishopCrawler()),
        ("mac-center", MacCenterCrawler()),
    ]

    session = get_session()
    try:
        for store_name, crawler in crawlers:
            products = crawler.crawl()  # type: ignore[union-attr]
            inserted = sum(1 for p in products if upsert_if_changed(session, p))
            typer.echo(f"{store_name}: {len(products)} products found, {inserted} inserted")
    finally:
        session.close()
```

Add `from apple_deals.db.session import get_session` to the top-level imports (it may already be there — check the existing imports before adding).

IMPORTANT: The lazy imports inside the function body (TiendasishopCrawler, MacCenterCrawler, upsert_if_changed) are intentional — they avoid importing playwright at module load time, which would fail if playwright is not yet installed. Keep them as lazy imports inside the function.

Do NOT change: the app = typer.Typer(...) line, the db_app sub-app, the @app.callback() main() function, or any other command stubs.
